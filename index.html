What Crate Pilot does

Crate Pilot is a desktop music library prep and organization tool built to save time. It scans and indexes your folders, helps you organize and export tracks by detected scale, and supports cleaner preparation workflows by renaming files using key and Camelot data.

Scans and indexes music folders Analyzes tracks for BPM, key, and Camelot Lets you browse and play tracks quickly Supports duplicate cleanup with clear review flow Includes rename and folder organization tools Designed for offline desktop usage

Product Tour

Each section below shows the matching Crate Pilot screen. Click any screenshot to open a larger view.

Overview

Main Workspace

The main view brings scan status, library browsing, player controls, and feature modules into one place.

This gives you a clear top-level workflow for moving from raw folders to a prepared collection.

Audio Analyzer

BPM, Key, and Camelot Analysis

The analyzer screen shows track-level musical data so you can make stronger preparation and selection decisions.

Use it to compare tracks faster and build more coherent sets.

Duplicate Cleaner

Duplicate Review and Cleanup

The duplicate screen helps identify likely duplicate files and inspect them before cleanup.

This reduces clutter and keeps your collection lean without blind deletion.

Folder Manager

Folder-Based Organization

Folder Manager can automatically create two destination folders, Maj and Min, for the selected source folder.

Crate Pilot then exports each song to the correct destination based on detected scale: major tracks go to Maj and minor tracks go to Min.

Rename Tracks

Consistent Naming Workflow

The rename screen applies repeatable patterns to filenames so your library is easier to sort and search.

This is especially useful when collections come from mixed sources with inconsistent naming.
